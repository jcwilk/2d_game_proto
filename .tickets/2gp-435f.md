# Implement centralized engine module for viewport and DisplayMode

Single module (e.g. src/engine.ts) owns Engine: DisplayMode, resolution policy, suppressHiDPIScaling. Plan §D.1.

## Design

Export width/height policy; **choose one** viewport policy for the repo: fixed logical resolution **or** native/backing resolution with scaling (plan §D.1)—document which in the module header. Set **`suppressHiDPIScaling`** explicitly to `true` or `false` with a one-line rationale (crisp pixels vs smooth scaling). No gameplay hardcodes canvas size outside module §D.3.

## Acceptance Criteria

1) `src/engine.ts` (or agreed path) documents factory/config pattern **and** names the chosen viewport policy and `suppressHiDPIScaling` behavior. 2) `DisplayMode` choice is explicit in code (enum value visible). 3) No stray `800x600`-style literals in `src/` outside this module (tests exempt).
