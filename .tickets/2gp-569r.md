# Configure Vite base and VITE_BASE for project GitHub Pages URLs

Project site `https://<user>.github.io/<repo>/` requires asset URLs under `/<repo>/`. **Normative:** **`.cursor/plans/project-implementation-deep-dive.md`** §A.2.1.

## Design

`vite.config.ts` reads **`import.meta.env.VITE_BASE`** (or equivalent) for `base`; **`VITE_BASE`** is a **public** path prefix—**not** a secret (contrast §E.0).

## Acceptance Criteria

1) With `VITE_BASE=/<repo>/`, `npm run build` emits HTML/JS that reference assets under that prefix; spot-check `dist/`. 2) **`README.md`** documents local `npm run dev` using **`base: '/'`** or the project’s chosen dev ergonomics. 3) **`vite.config.ts`** comment or **`README.md`** (≤5 lines) cites §A.2.1.
